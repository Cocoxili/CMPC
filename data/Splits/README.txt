This folder contains test files to generate the train/val/test split used in the paper:
Nagrani, Arsha, Samuel Albanie, and Andrew Zisserman. "Learnable PINs: Cross-Modal Embeddings for Person Identity." ECCV, 2018.


For each file, the format is: $name $set, where set can take one of the following values:

1 = train 
2 = val 
3 = seen-heard test 
4 = unseen-unheard test 


Files present in this folder are:

identity_splits.txt 
face_set_labels.txt 
voice_set_labels.txt
